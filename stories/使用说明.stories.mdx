# 使用说明
[官网地址](https://storybook.js.org/)

## yarn
`-D 开发依赖` `-W 工作区` `-D -W 把依赖安装到工作区的根目录`

## 初始化storybook
    npx sb init

## 安装依赖vue3.0
    cnpm install vue@next --save
    yarn add vue vue-template-compiler vue-loader --save-dev

## 运行storybook
    yarn storybook

## yarn workspaces
* `管理包的依赖`

## lerna 
* `使用lerna管理npm和git的发布`
* `发布前需要把yarn的源换成原来的,不能使用淘宝源发布`
* `发布前需要检查是否已提交代码(只提交 publish操作交给lerna)`

    ##### 安装
        yarn global add lerna

    #### 查询当前源
        yarn config get registry 
        npm config get registry 

    #### 还原镜像
    yarn config set registry https://registry.npmjs.org/ <br />
    npm config set registry https://registry.npmjs.org/

    ##### 淘宝源
        yarn config set registry https://registry.npm.taobao.org/
        npm config set registry https://registry.npm.taobao.org/

    #### 查询当前登录的用户名
        npm whoami 

    #### 登录npm
        npm login

    ##### 发布 
        lerna publish 

## plop
* `自动构建组件`

        yarn add plop -W -D
