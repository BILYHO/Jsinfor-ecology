# 使用说明

## yarn
`-D 开发依赖` `-W 工作区` `-D -W 把依赖安装到工作区的根目录`

## 运行storybook
    yarn storybook

## 安装依赖
- 开发依赖
   
        yarn add xxxx -D -W

- 给某个组件安装单独的依赖
    
    `jsi-bar不是文件夹名称,而是对应组件包内packjson的name`
    
        yarn workspace jsi-bar add echarts

## plop 
`自动生成组件文件夹以及文件夹内所需的文件`
    
    yarn plop

## lerna 
* `使用lerna管理npm和git的发布`
* `发布前需要把yarn的源换成原来的,不能使用淘宝源发布`
* `发布前需要检查是否已提交代码(只提交 publish操作交给lerna)`

    ##### 安装
        yarn global add lerna

    #### 查询当前源
        yarn config get registry 
        npm config get registry 

    #### 还原镜像
    yarn config set registry https://registry.npmjs.org/ <br />
    npm config set registry https://registry.npmjs.org/

    ##### 淘宝源
        yarn config set registry https://registry.npm.taobao.org/
        npm config set registry https://registry.npm.taobao.org/

    #### 查询当前登录的用户名
        npm whoami 

    #### 登录npm
        npm login

    ##### 发布 
        lerna publish
